# wurm
Python simulation of C. elegans, via 302 neurons and reinforcment learning.

## Worm Stimuli
- Light reception
- Vibration sensation
- Food sensation

## Worm Actions
- Grip head/tail
- Flex muscle segment longitudinally (contract/extend)
- Flex muscle segment transversely (left/right)

## RL
- Proximal Policy Optimization (PPO)
- Deep Q Network (DQN)
